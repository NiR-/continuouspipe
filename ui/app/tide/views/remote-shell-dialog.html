Open a bash shell in the pod '{{endpoint.name}}'

cp-remote bash -i -e {{environment.identifier}} -s {{endpoint.name}} -f {{environment.flow.uuid}}

If /bin/bash is not available on the pod use

cp-remote exec -i -e {{environment.identifier}} -s {{endpoint.name}} -f {{environment.flow.uuid}} -- /bin/sh

If you have a Remote Development Environment initialised on the same flow, you can omit the --interactive (-i) flag

cp-remote bash -e {{environment.identifier}} -s {{endpoint.name}} -f {{environment.flow.uuid}}
